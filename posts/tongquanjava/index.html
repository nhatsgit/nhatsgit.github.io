Lập Trình Java Cơ Bản: Hướng Dẫn Dành Cho Người Mới Bắt Đầu

Dec 30, 2024 · 4 mins read
Java Programming Basics Learning Java

Java Là Gì?

Java là một ngôn ngữ lập trình phổ biến, mạnh mẽ và độc lập nền tảng. Với Java, bạn có thể phát triển nhiều loại ứng dụng khác nhau, từ ứng dụng di động, web, đến hệ thống nhúng.

Dưới đây là các khái niệm cơ bản nhất khi học Java dành cho người mới bắt đầu:

1. Cài Đặt Java

Để bắt đầu, bạn cần:

  • Cài đặt JDK (Java Development Kit) từ trang web chính thức của Oracle hoặc OpenJDK.
  • Sử dụng IDE như IntelliJ IDEA, Eclipse, hoặc NetBeans, hoặc viết code trong một trình soạn thảo đơn giản như VSCode.

2. Cấu Trúc Chương Trình Java

Mỗi chương trình Java bắt buộc phải có ít nhất một lớp và phương thức main(). Đây là điểm bắt đầu của mọi ứng dụng Java.

Ví dụ: Hello World

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Chào mừng bạn đến với lập trình Java!");
  }
}
  • public class HelloWorld: Khai báo lớp có tên HelloWorld.
  • public static void main(String[] args): Phương thức chính, nơi chương trình bắt đầu thực thi.
  • System.out.println(...): In chuỗi ra màn hình.

3. Biến và Kiểu Dữ Liệu

Java hỗ trợ nhiều kiểu dữ liệu cơ bản như:

  • int: Số nguyên
  • double: Số thực
  • char: Ký tự
  • boolean: Giá trị đúng hoặc sai

Ví dụ:

public class VariablesExample {
  public static void main(String[] args) {
    int age = 25;
    double salary = 55000.75;
    char grade = 'A';
    boolean isJavaFun = true;

    System.out.println("Tuổi: " + age);
    System.out.println("Lương: " + salary);
    System.out.println("Hạng: " + grade);
    System.out.println("Java có thú vị không? " + isJavaFun);
  }
}

4. Vòng Lặp và Câu Lệnh Điều Kiện

  • Câu lệnh điều kiện: if-else giúp kiểm tra điều kiện và thực thi mã tương ứng.
  • Vòng lặp: for, while, và do-while được sử dụng để lặp đi lặp lại các đoạn mã.

Ví dụ:

public class ControlFlowExample {
  public static void main(String[] args) {
    int number = 7;

    // Kiểm tra số chẵn hay lẻ
    if (number % 2 == 0) {
      System.out.println(number + " là số chẵn.");
    } else {
      System.out.println(number + " là số lẻ.");
    }

    // Vòng lặp in các số từ 1 đến 5
    for (int i = 1; i <= 5; i++) {
      System.out.println("Số: " + i);
    }
  }
}

5. Phương Thức (Methods)

Phương thức là khối mã thực hiện một nhiệm vụ cụ thể và có thể tái sử dụng.

Ví dụ:

public class MethodsExample {
  // Phương thức tính tổng hai số
  public static int sum(int a, int b) {
    return a + b;
  }

  public static void main(String[] args) {
    int result = sum(10, 20);
    System.out.println("Tổng: " + result);
  }
}

6. Lập Trình Hướng Đối Tượng (OOP)

Java là ngôn ngữ hướng đối tượng. Các khái niệm chính:

  • Lớp (Class): Mẫu thiết kế cho đối tượng.
  • Đối tượng (Object): Thực thể của lớp.
  • Kế thừa (Inheritance), Đa hình (Polymorphism), Đóng gói (Encapsulation).

Ví dụ:

class Animal {
  String name;

  public void eat() {
    System.out.println(name + " đang ăn.");
  }
}

public class OOPExample {
  public static void main(String[] args) {
    Animal cat = new Animal();
    cat.name = "Mèo";
    cat.eat();
  }
}

7. Tại Sao Nên Học Java?

  • Nhiều cơ hội nghề nghiệp: Java được sử dụng trong nhiều lĩnh vực như phát triển web, ứng dụng di động (Android), và hệ thống doanh nghiệp.
  • Thư viện phong phú: Java cung cấp rất nhiều thư viện và framework hỗ trợ.
  • Cộng đồng lớn: Dễ dàng tìm kiếm tài liệu và sự trợ giúp.

Kết Luận
